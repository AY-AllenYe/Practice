# 49. 字母异位词分组

## version 1

​	哈希表

​		先排序（按照a-z重新组合单词，作为键。异位词的重排一定是一样的）

​		再利用哈希表输出键对应的词汇组。



​	time: $O(n*k*log(k))$
​	space: $O(n*k)$

## version 2

​	哈希表

​		先计数，登记每个字母出现频次

​		再合并频次，生成字母出现频次的字符串（用分隔符隔开，否则会错位），作为键

​		最后利用哈希表输出键对应的词汇组。



time: $O(n*(k+n))$
space: $O(n*k)$
